RunInputAsync(DeviceInputCommand, CancellationToken)

Sends an input command to the first feature that supports the command's input type. For Read commands, returns the reading. For Subscribe/Unsubscribe, returns null.

Declaration
public Task<InputReading> RunInputAsync(DeviceInputCommand command, CancellationToken token = null)
Parameters
Type Name Description
DeviceInputCommand command

The input command to send.
CancellationToken token

Cancellation token.
Returns
Type Description
Task<InputReading>

The input reading for Read commands, null for Subscribe/Unsubscribe.
Examples
// Read battery level
var reading = await device.RunInputAsync(DeviceInput.Battery.Read());
var batteryLevel = reading?.GetValue(InputType.Battery);

// Subscribe to button events
await device.RunInputAsync(DeviceInput.Button.Subscribe());
